    • npm并行多个命令

    遇到一个这样的问题，使用create-react-app搭建项目时，加入了sass，这是需要实时监听sass文件，将其转化为css文件
    于是npm run start启动服务后，需要在终端再次执行一条命令 npm run watch-css

    这是我忍受不了的

    事实上，我希望一条命令可以同时执行下面两条命令

    解决方法

    1.安装 npm-run-all
    2.添加一条新的命令

    此时，npm run dev便同时会执行 npm run watch-css 和 npm run start
    完美

    npm-run-all

    • run-s 运行串联任务
    • run-p 运行并行任务
